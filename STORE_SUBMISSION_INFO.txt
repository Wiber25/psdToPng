[Chrome 웹 스토어 게시 정보]

1. 전용 목적 (Single Purpose)
이 확장 프로그램의 주 목적은 사용자가 별도의 무거운 디자인 소프트웨어 설치 없이 브라우저 내에서 PSD(Photoshop) 파일을 PNG 이미지로 즉시 변환할 수 있도록 하는 것입니다. 사용자는 파일을 드래그 앤 드롭하여 간편하게 변환하고 결과물을 다운로드할 수 있습니다.

2. 권한 사용 근거 (Permission Justification)

- storage:
사용자의 라이선스 인증 상태(Pro 모드 활성화 여부)와 사용자 이메일 정보를 로컬 브라우저에 저장하여, 확장 프로그램을 다시 열었을 때도 인증 상태를 유지하기 위해 사용합니다. 서버와의 인증 세션을 유지하기 위한 필수적인 데이터 저장소로 활용됩니다.

3. 원격 코드 (Remote Code)
- 아니요 (No)
(설명: Firebase Firestore와 통신하기 위해 fetch API를 사용하지만 외부 스크립트를 동적으로 로드하거나 실행하지 않습니다.)

4. 사용자 데이터 (User Data)

- 수집하는 데이터: 개인 식별 정보 (이메일 주소)
- 사용 목적: 제품 기능 (Product feature) - 라이선스 인증 및 Pro 기능 잠금 해제
- 명시적 동의: 사용자가 라이선스 등록 화면에서 직접 이메일을 입력하고 인증 버튼을 누름으로써 수집됩니다.
